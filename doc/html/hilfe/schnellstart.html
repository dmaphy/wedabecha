Kurzanleitung für WeDaBeCha

Werkzeug zur grafischen Darstellung und Beschriftung von Charts

Schnellstart:

 

1. Klicken sie auf "Tabelle importieren"

2. in der ersten Zeile klicken sie auf den "Öffnen"-Button

3. im Dialogfeld "Tabelle 1 für den Import vorbereiten" klicken sie auf "Durchsuchen"

4. wählen sie eine Datei, die sie importieren möchten, öffnen sie sie

5. wieder zurück im Dialogfeld "Tabelle 1 für den Import vorbereiten" klicken sie auf "Datum"

6. stellen sie das gewünschte Datumsformat ein und klicken sie auf OK

7. im Dialogfeld "Tabelle 1 für den Import vorbereiten" klicken sie auf OK

8. auch im Dialog "Tabellen importieren" klicken sie auf OK

 

Fertig. Sie haben ihre erste Kurve mit WeDaBeCha erstellt. Herzlichen Glückwunsch.
