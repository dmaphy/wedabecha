Entwickler-Version

Um auf dem aktuellsten Stand der Entwicklung zu bleiben laden Sie sich die Quelltexte aus dem Git-Repository. Hiermit haben Sie eine Grundlage um Modifizierungen etc. vorzunehmen. Wir empfehlen nicht, die Entwickler-Version produktiv einzusetzen.

Anonym an die aktuellen Quelltexte kommen Sie über den Befehl
git clone git://github.com/dmaphy/wedabecha.git.

Alternativen und eine Möglichkeit, direkt Online durch die Quelltexte zu browsen finden Sie direkt auf der Github-Seite von wedabecha.
