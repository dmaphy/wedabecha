Willkommen auf der Projektseite von wedabecha,
einem Werkzeug zur Darstellung und Bearbeitung von Charts von Aktienkursen und anderen zeitabhängigen Daten.

über wedabecha

Bei wedabecha handelt es sich um eine Software welche während eines Studienprojektes entstanden ist. Ziel des Projektes ist es, ein Werkzeug zu sein, welches

  • Daten von Aktienkursen in bestimmten Formaten einlesen kann (CSV, TAB-Separated, u.a.)
  • aus diesen Daten grafische Kurven mit verschiedenen Optionen zeichnet
  • eine Möglichkeit bietet, Linien, Pfeile und Annotationen vorzunehmen

Schon während der Entwicklung in der Studienzeit haben sich die vier Entwickler Robert, Matthias, Martin und Dominic dazu entschlossen das Projekt als OpenSource zu entwickeln, so dass es was das Hosting und Versionsverwaltung angeht bis 2011 bei berliOS.de verwaltet wurde. Mit der Nachricht der Schließung von BerliOS in 2011 wurde das Projekt dann auf die Plattform GitHub umgezogen.

Seit dem ersten Release im Februar 2005 und damit dem Abschluss des Studienprojektes wird wedebecha komplett als OpenSource weiterentwickelt.

geschichte

  • Oktober 2011

    wedabecha wechselt nach der Nachricht über die Schließung von BerliOS auf die Versionsverwaltung Git und die Plattform GitHub

  • März 2007

    Das Projekt ist von nun an unter der Adresse wedabecha.org erreichbar

  • Oktober 2004 bis Februar 2005

    Entwicklung der Software während als Studienprojekt

herunterladen

Unter wedabecha.org/downloads finden Sie immer die aktuellste Version der Software inklusive:

  • Binärversion mit Installer für einfachsten Komfort
  • Dokumentation und Beispieldateien
  • Quelltexte der aktuellen stabilen Version
  • und Quelltexte der instabilen Version aus dem Subversion-Repository

mitmachen

Auf den Projektseiten können Sie auch den Entwicklungsstatus mitverfolgen, neue Vorschläge für die Software machen oder sich selber als Entwickler bewerben.
Was Sie für wedabecha tun können?

  • direkt als Entwickler in der Programmierung einsteigen, Vorraussetzung: Sie können Java programmieren
  • Dokumentationen schreiben und vervollständigen, Vorraussetzung: Sie sind der deutschen und/oder englischen Sprache mächtig
  • die Software und/oder Dokumentation übersetzen, Vorraussetzung: Sie sind der deutschen Sprache und der Übersetzungssprache mächtig
  • Spenden für den guten Zweck: einige Entwickler haben ein PayPal-Konto an das Sie beliebige Beträge überweisen können

Wenn Ihnen auch nur einer der genannten Punkte zusagt sind Sie richtig bei uns!
