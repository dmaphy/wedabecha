Sie brauchen Hilfe?

Sie haben ein Problem oder brauchen Hilfe mit dem Programm?
Eventuell hatte auch schon jemand ähnliche Probleme. Eine Datenbank finden Sie unter http://developer.berlios.de/support/?group_id=2470.
Unter dieser Adresse können Sie auch nach Support speziell für ihr Problem fragen.

Vorschlag für eine neue Funktion?

Sie haben eine Idee für neue Funktionen im Programm, neue Features? Unter http://developer.berlios.de/feature/?group_id=2470 finden Sie das System um "Feature Requests" anzufragen.

Fehler im Programm?

Sie haben einen sogenannten Bug im Programm gefunden? Fühlen Sie sich frei uns einen Bugreport zu schreiben. Extra für diesen Zweck finden Sie unter http://developer.berlios.de/bugs/?group_id=2470 ein System, wo eventuell auch schon Ihr Problem und eine Lösung beschrieben ist.

Probleme? Fragen? Anregungen?

Für alle Anliegen im Zusammenhang mit wedabecha dürfen Sie sich im Support-Forum aussprechen.
